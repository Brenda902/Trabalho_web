Bem vindo(a) a biblioteca!

A biblioteca funciona da seguinte forma: o conteúdo foi catalogado de acordo com as doenças/mal estar encontradas nas fontes usadas para o site, logo para saber qual planta medicinal se encaixa para o seu tratamento clique na primeira letra de seu respectivo mal estar/doença, etc. Você será direcionado para uma página com as doenças e as plantas recomendadas para tratá-las.
